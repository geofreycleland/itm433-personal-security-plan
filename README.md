# ITM 433 – Personal Security Plan (PSP)

## Overview
This project presents a simple, user-friendly Personal Security Plan (PSP) designed to help individuals organize critical security information, reduce personal cybersecurity risk, and prepare a trusted helper to assist during emergencies. The plan emphasizes clarity, practicality, and real-world usability over technical complexity.

## Project Objectives
- Help users identify and prioritize their most important personal and digital assets  
- Promote practical, everyday security behaviors that reduce account takeover and identity theft risk  
- Provide a structured, easy-to-complete plan suitable for printing and offline storage  
- Translate cybersecurity best practices into guidance accessible to non-technical users  

## Course Information
- **Course:** ITM 433  
- **Project Type:** Master’s-level course project  

## Intended Audience
This Personal Security Plan is designed for the general public, including individuals with limited technical background. It assumes no prior cybersecurity knowledge and focuses on clear language, guided explanations, and actionable steps.
